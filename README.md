# HEARTHHOLD — 3D survival city-builder (work in progress)

Build by day, defend by night — instant, no waiting timers. Real-time 3D in the browser (Three.js).

**Current: 3D look test** → https://matthewhalle09-jpg.github.io/hearthhold/

3D engine: [Three.js](https://threejs.org) (MIT, vendored). World is procedural low-poly; art will expand with CC0 model packs (Kenney / Quaternius / Poly Pizza).
